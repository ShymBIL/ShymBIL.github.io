А - Нурсултан и игра с числами

В этой задаче нужно было найти сколько чисел из массива a(a1, a2 ... an) делятся на к и ещё посчитать ихнюю сумму. Для решения, давайте создадим col и cnt. Проверям каждый элемент массива, если ai % k == 0 , то делаем col++, cnt += a[i]. После проверки тупо выводим col и cnt(cnt - long long int - потому что если к будет равен 1 и a1 = a2 .... = an = 10^9 то ответ будет 10^9 * 10^5(максимальное ограничение размера массива) = 10^14 а это не помешяется int).

C - Нысан и Магазины

В этой задаче нужно просто проверить каждый элемент массива a(a1, a2 ... an). Создадим long long int money = 0(вы наверное уже догадались почему long long int) - денги которые Нысан cможет украсть. Если мы оказались на позиции, где a[i] >= 0 то money += a[i]. Если же мы оказализ на позиции в котором a[i] = -1 и a[i + 1] = -1 то уже Нысан не может идти на другие магазины(В i-ый магазин можго добраться толька с i-1 и i-2) и из за этого мы выводим сумму денег.

E - Задача Нархана

В последней задаче нужно посчитать количество четных и нечетных чисел. Если количества четных больше чем нечетных выведите все четные числа, если меньше выведите все нечетные числа. Если количество равны, сравните по суммам. То есть, если сумма четных больше чем сумма нечетных выведите все четные числа, если меньше выведите все нечетные числа
